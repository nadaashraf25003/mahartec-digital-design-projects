# Digital Design Projects – Verilog (Mahartec Course)

This repository contains a collection of digital design modules implemented in Verilog HDL as part of the MaharTec training program. The projects cover combinational and sequential logic design, counters, ALU design, and display systems.

---

## 📌 Projects Included

### 1. Combinational Circuits
- 2:1 Multiplexer
- ALU (Arithmetic Logic Unit)
- 7-Segment Decoder

### 2. Sequential Circuits
- D Flip-Flop (async reset)
- Johnson Counter
- One-Hot Counter FSM
- BCD Counter (normal, flexible, FSM-based)

### 3. Clock & Timing
- Clock Divider (1Hz generator)

### 4. Display Systems
- BCD to 7-Segment Display
- Counter to Decoder System
- Multi-display ALU output system

### 5. Integrated Systems
- Top Keypad ALU Display System
- Seconds Counter Top Module


---

## ⚙️ Tools Used
- Vivado / ModelSim
- Verilog HDL
- FPGA Simulation

---

## 🎯 Learning Outcomes
- Digital logic design
- FSM design
- Counter design techniques
- Display interfacing
- Modular hardware design
- System integration
